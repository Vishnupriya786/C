# Pre and Post Increment

```
#include<stdio.h>
int main ()
{
	int Number = 3;
	printf("Original number is %d\n",Number);
  printf("The value of post increment number is %d\n",Number++); // Post Increment
  Number = 3;
	printf("The value of pre increment number is %d\n",++Number); // Pre Increment
}
```

```
OUTPUT:
Original number is 3
The value of post increment number is 3
The value of pre increment number is 4
```
