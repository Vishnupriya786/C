# Arithmetic Operator

```
#include<stdio.h>
int main ()
{
	int Number1,Number2;
	printf("Enter first number:");
	scanf("%d",&Number1);
	printf("Enter second number:");
	scanf("%d",&Number2);
	printf("The addition of %d and %d is %d\n",Number1,Number2,Number1+Number2); // addition
	printf("The subtraction of %d and %d is %d\n",Number1,Number2,Number1-Number2); // subtraction
	printf("The multiplication of %d and %d is %d\n",Number1,Number2,Number1*Number2); // multiplication
	printf("The division of %d and %d is %d\n",Number1,Number2,Number1/Number2); // division
	return 1;
}
```

```
OUTPUT:
Enter first number:10
Enter second number:5
The addition of 10 and 5 is 15
The subtraction of 10 and 5 is 5
The multiplication of 10 and 5 is 50
The division of 10 and 5 is 2
```
