# Relational Operators

```
#include<stdio.h>
int main ()
{
	int Number1 = 3, Number2 = 4;
	printf("%d\n",Number1 == Number2); // 3 == 4 
	printf("%d\n",Number1 < Number2); // 3 < 4 
	printf("%d\n",Number1 > Number2); // 3 > 4 
	printf("%d\n",Number1 <= Number2); // 3 <= 4 
	printf("%d\n",Number1 >= Number2); // 3 >= 4 
	printf("%d\n",Number1 != Number2); // 3 != 4 
	// 0 ---> False, 1 ---> True 
}
```

```
OUTPUT:
0
1
0
1
0
1
```
