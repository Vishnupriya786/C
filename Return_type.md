# Return type

```
#include<stdio.h>
int main () // int is used to return value and void is used when return value doesn't required.
{
	int Number1 = 4;
	int Number2 = 5;
	printf("The addition of %d and %d is %d",Number1,Number2,Number1+Number2);
	return 1;
	
}
```

```
OUTPUT:
The addition of 4 and 5 is 9
--------------------------------
Process exited after 0.03784 seconds with return value 1
```
