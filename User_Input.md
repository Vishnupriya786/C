# User Input

```
#include<stdio.h>
int main ()
{
	int Number1,Number2;
	printf("Enter first number:");
	scanf("%d",&Number1);
	printf("Enter second number:");
	scanf("%d",&Number2);
	printf("The addition of %d and %d is %d",Number1,Number2,Number1+Number2);
	return 1;
}
```

```
OUTPUT:
Enter first number:5
Enter second number:4
The addition of 5 and 4 is 9
```
